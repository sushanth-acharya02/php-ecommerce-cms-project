# 🛒 PHP Ecommerce CMS Project

## 📌 Description
This is a full-featured eCommerce web application developed using PHP and MySQL.  
It includes both user-side functionality and an admin panel (CMS) for managing products, categories, and orders.

---

## 🚀 Features
- User Registration & Login
- Product Listing
- Admin Panel (CMS)
- Add/Edit/Delete Products
- Category Management
- Image Upload

---

## 🛠️ Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP

---

## 📂 Project Structure
- ecommerce/
  - cms/ (Admin Panel)
  - css/
  - js/
  - images/
  - uploads/

---

## ▶️ How to Run Project
1. Install XAMPP
2. Copy project folder into `htdocs`
3. Start Apache & MySQL
4. Import database (.sql file)
5. Open browser and run:
   http://localhost/ecommerce
